מדריך למשתמש:
מעבר בין הפקדים בעזרת tab
לחיצה על כפתורים בעזרת enter ו space
check list וradio box - ניווט בין האפשרויות עם המקשים למעלה ולמטה, בחירה בעזרת enter ו space
numeric box - מורכב משני כפתורים "-", "+", לחיצה בעזרת enter ו space
combo box - פתיחת כפתור אפשרויות בעזרת enter ו space, ניווט בין האפשרויות עם המקשים למעלה ולמטה
מצורפת תוכנית main העושה שימוש בכלל הפקדים ומדמה טופס הגשת מועמדות לעבודה
לחיצה על כפתור submit מייצרת טופס המכיל את הפרטים שהזין המשתמש
בפתיחת message box בלחיצת על כפתור OK תאפשר הגשת טופס נוסף ולחיצה על כפתור Ctl תסיים את ריצת התוכנית
